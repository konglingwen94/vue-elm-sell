## 安装依赖

```
npm install
```

### 开启一个具有热重载功能的开发服务器

```
npm run serve
```

### 构建项目

```
npm run build
```

## 主要功能

- [x] 商品菜单联动导航展示
- [x] 添加移除商品到购物车
- [x] 购物车商品列表
- [x] 清空购物车
- [x] 确认支付提示
- [x] 查看商品详情
- [x] 商品评价
- [x] 店铺综合评论
- [x] 商家信息展示
- [x] 商家收藏
- [x] 商家详情弹窗
- [ ] 商品滚动列表菜单标题吸顶
- [ ] 页面布局优化
- [ ] 添加商品到购物车时的动画效果
